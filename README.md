# 명운서재 MVP

전통 명리 해석과 AI 기술을 활용해 사용자의 고민에 맞는 행동 방향을 제안하는 모바일 우선 PWA 데모입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 데모 범위

- 고민 선택 → 생년월일/현재 고민 입력 → 무료 개인화 해석
- 유료 상품 3종 비교, `FIRST20` 쿠폰, 카카오 로그인/결제 데모
- 상세 결과 생성 연출 및 결과 카드
- 브라우저 인쇄 기반 PDF 저장
- 결과/구매 인증 후기 로컬 저장
- `/admin` 관리자 지표·상담 결과·후기 확인
- PWA manifest 및 서비스 워커

카카오 OAuth, 실결제, 서버 DB, 외부 AI 모델 호출은 운영 키가 필요한 경계로 분리해 데모 UX만 구현했습니다.
